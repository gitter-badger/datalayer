Datalayer Notebook for Big Data Scientists to utilize Microsoft Account and Microsoft Azure Active Directory Identities

16 April 2016

PRESS RELEASE

BRUSSELS, BE – 16 APRIL 2016 08:00 CET – Datalayer today announced the integration of the Datalayer WEB Notebook for Big Data Scientists with Microsoft OneNote. The company also announced that Datalayer WEB Notebook will be deployed on Microsoft Azure.

This integration is available for Windows Live, as Office 365 users via the Datalayer WEB Notebook. This authentication can be used today by the Data Scientists to publish their data analysis in the Microsoft OneNote online service, more easily read and accessible to the Business stakeholders.

“Microsoft Azure brings the power of deploying and scaling our WEB Notebook, but overall offers us the opportunity to build on a user ecosystem, giving them access to an extending range of Microsoft on line services,“ said Eric Charles, Founder of Datalayer.

The availability of publication to Microsoft OneNote will be extended in function of the Microsoft API availability and driven by user needs.

“Datalayer's offering is bridging the gap between science and business, and fosters business communication. Utilizing Microsoft Azure, Datalayer is offering their customers the opportunity to better communicate and work with their data-driven strategy,” said Nicole Herskowitz, Senior Director of Product Marketing, Microsoft Azure, Microsoft Corp.

You can register as from today on http://datalayer.io as a Microsoft on-line user. Upon the Windows Live as the Office 365 authentication, professional packages are also available for on premise and Azure Active Directory deployments.

About Datalayer

Datalayer Notebook allows Data Scientists to play with Big Data, build innovative models, and share results easily on Microsoft on line services. Datalayer develops a WEB Notebook that allows the data scientist and business analyst to easily build models on internal and external big datasets.

The Notebook runs on open-source Spark/Hadoop, is available on Azure, and can also be deployed on-premises. Mathematical libraries, real-time collaboration, and publication into Microsoft OneNote make the Datalayer Notebook a first-choice solution for enterprises and academics working to process Big Data.
