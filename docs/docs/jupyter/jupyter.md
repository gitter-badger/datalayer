## Install

```
npm install -g configurable-http-proxy
sudo pip3 install --upgrade notebook
sudo pip3 install --upgrade jupyter
sudo pip3 install --upgrade jupyterhub
```

```
jupyter labextension install @jupyterlab/hub-extension
```
